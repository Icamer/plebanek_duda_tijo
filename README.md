# TiJO Projekt

Maly projekt zaliczeniowy z przedmiotu "Testowanie i jakosc oprogramowania".

Program sortujacy tablice n-elemnotowa.

## Getting Started


### Prerequisites

Aby zainstalowas

```
git clone https://github.com/Icamer/plebanek_duda_tijo
```

### Installing

Instalacja bibliotek:

```
npm install
```

## Running the tests

Aby uruchomic testy:

```
npm test
```

```
npm run stryker
```

```
npm run jshint
```

```
npm run coverage
```
